Tecnologias

As tecnologias utilizadas foram HTML5, CSS3 e JavaScript. As tags semânticas de HTML5 foram usadas. Os estilos usando CSS3 foram aplicados segundo as orientações do W3C.

O editor de textos usado foi o Sublime Text. O editor de imagens usado foi o GIMP, uma alternativa ao Photoshop, porém é software livre. O sistema operacional usado no desenvolvimento foi o Linux.
